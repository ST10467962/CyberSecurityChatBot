# Cybersecurity Awareness Chatbot

## Project Overview
This project is a Cybersecurity Awareness Chatbot developed in C# using Windows Forms.

The chatbot helps users learn about:
- Password safety
- Phishing 
- Privacy protection


The chatbot includes:
- GUI interface
- Voice greeting
- Sentiment detection
- Memory recall
- Keyword recognition
- Random responses
- Error handling
 

---

## Technologies Used
- C#
- Windows Forms
- Visual Studio 2022
- GitHub
- GitHub Actions

---

## How to Run the Project
1. Open the solution in Visual Studio 2022.
2. Build the project.
3. Run the application.
4. Interact with the chatbot through the GUI.

---

## Features
- Personalized greetings
- Cybersecurity tips
- Conversation flow
- User memory
- Sentiment analysis
- Input validation
- Voice greeting using WAV audio
